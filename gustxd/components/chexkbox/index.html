Checkbox

Checkboxes allow the user to select one or more items.

If we wanted to present more information than what a label woul allow, using this detail section will size the content correctly.

import { Checkbox, CheckboxFieldset } from "@labxd/gustxd";
...
<CheckboxFieldset sr="checkbox-group">
    <Checkbox
        id="default-0"
        label="I can see the difference">
              If we wanted to present more information than what a label woul allow,
              using this detail section will size the content correctly.
    </Checkbox>
</CheckboxFieldset>
