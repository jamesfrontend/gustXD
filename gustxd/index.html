gustXD
Open Storybook

A React component library built with Tailwind CSS. It is meant to work with your tailwind setup. Install, point, and start using the components.

Installation​

npm install @labxd/gustxd

Add our preset​

The preset will merge with your configuration, not override them. Point tailwind to our components.

// tailwind.config.js

module.exports = {
  presets: [require("@labxd/gustxd/tailwind")],
  content: [
    "node_modules/@labxd/gustxd/dist/**/*.{js,ts,jsx,tsx}",
    ...
  ],
};
