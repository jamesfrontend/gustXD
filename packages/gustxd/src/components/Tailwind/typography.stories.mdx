## Heading (Red Hat Display)

Red Hat Display is the default font used for all headers. You do not need to add the class, but if you want to be explicit:

<Canvas>
  <Story name="Heading (Red Hat Display)">
    <main className="space-y-1">

      <h1 className="font-display font-semibold">font-display font-semifont</h1>
      <h1 className="font-display font-bold">font-display font-bold</h1>
    </main>
  </Story>
</Canvas>
